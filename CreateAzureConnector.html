Создание Azure Connector

Для разворачивания среды в рамках Lcs проекта, прежде всего, необходимо создать Azure Connector.

Azure connector позволяет использовать azure подписку в определенном Lcs проекте.

Для создания azure connector необходимо выполнить следующие шаги:

  1. Перейти в Lcs проект.
  2. Привязать Azure DevOps к текущему проекту (этот шаг не обязателен, но необходим для разворачивания среды).
  3. Ввести имя для azure connector. Также следует ввести ID azure подписки и установить переключатель «Configure to use Azure Resource Manager (ARM)» в положение Yes. В появившемся полее ввести домен, указанный в подписке Azure.
  4. Далее необходимо добавить специального пользователя Dynamics Deployment Services в подписку azure. Убедитесь, что права пользователя установлены как Contributor.
  5. После добавления пользователя перейдите на вкладку Lcs и нажмите Next. Убедитесь, что установлена галка.
  6. После нажатия Next появится меню, где необходимо скачать сертификат. Сертификат можно создать в Azure по пути Your subscription -> Management Sertificates -> upload - > указать скачанный сертификат.
  7. После нажатия Next, необходимо выбрать регион, в котором будут развернуты компоненты azure. Рекомендуется выбирать North Europe.
  8. Убедитесь, что azure connector создан и установлена галка ARM enabled.
